# ai-powered-hr-chatbot
AI-Powered HR Assistant for Nestlé — A conversational chatbot built with Python, LangChain, FAISS, OpenAI GPT-3.5 Turbo, and Gradio. This project demonstrates document-based question answering by extracting HR policy information, embedding it into a FAISS vector store, and serving responses through a user-friendly Gradio interface.
